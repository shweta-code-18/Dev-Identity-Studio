# Dev Identity Studio

A premium developer identity card generator built with HTML, CSS, and JavaScript.

## Features Included

- Fully responsive glassmorphism interface
- Sticky responsive header
- Dark and light mode with local theme persistence
- Modern developer card generator (name, role, skill chips, GitHub + LinkedIn links)
- Custom profile image upload
- Automatic GitHub avatar fetch from username or profile URL
- 3D tilt interaction on desktop pointers
- Animated particle background
- Download generated card as PNG
- Copy generated card details to clipboard
- Randomize card accent style
- Form reset after each card generation
- Download fallback when CDN capture library is unavailable
- Responsive footer section

## Usage

1. Fill in your profile fields.
2. Enter a GitHub username or profile URL.
3. Optionally upload a custom image.
4. Click `Generate Identity`.
5. Click `Download Card` to save the generated card image.
6. Use `Ctrl/Cmd + Shift + D` as a quick download shortcut.
